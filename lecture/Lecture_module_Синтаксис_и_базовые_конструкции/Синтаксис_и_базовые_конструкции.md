## [translate:1. Переменные и типы данных]
[translate:В C# используются строгие типы данных — вся информация, обрабатываемая программой, принадлежит определённому классу типов:]
- [translate:int — целое число]
- [translate:double — число с плавающей точкой]
- [translate:string — строка]
- [translate:bool — логический тип]
[translate:Переменные объявляются явно с указанием типа, например:]

```csharp
int age = 25;
double price = 99.5;
string name = "Ivan";
bool isActive = true;
```

## [translate:2. Константы и преобразования типов]
[translate:Константа объявляется с ключевым словом const:]

```csharp

```
[translate:Типы преобразуются явно (cast):]

```csharp
int x = (int)99.99;
string y = x.ToString();
```
