# Массивы и коллекции в C#

В этом модуле разбираются ключевые структуры данных языка C#, а также методы работы с ними: инициализация, индексация, перебор и алгоритмы сортировки.

## Массивы

Массив — это структура для хранения фиксированного набора элементов одного типа. В C# массив объявляется следующим образом:

```csharp
int[] numbers = new int[5]; // массив из 5 элементов

// Инициализация массива при объявлении
int[] arr = {1, 2, 3, 4, 5};

// Индексация массива
int first = arr[0]; // 1
int last = arr[arr.Length - 1]; // 5
```

### Многомерные массивы

Многомерные массивы позволяют формировать матрицы и таблицы данных:

```csharp
int[,] matrix = new int[3, 3]; // двумерный массив 3x3
matrix[0, 1] = 7; // присвоение значения
```

## Коллекции .NET

Коллекции представляют собой динамические контейнеры данных:
- `List<T>` — динамический список
- `Dictionary<TKey, TValue>` — ассоциативный массив (словари)
- `Stack<T>` — стек
- `Queue<T>` — очередь

**Примеры:**
```csharp
List<string> names = new List<string>();
names.Add("Ivan");
names.Add("Anna");
string second = names[1]; // "Anna"

Dictionary<string, int> ratings = new Dictionary<string, int>();
ratings["Олег"] = 5;
ratings["Мария"] = 8;
int mark = ratings["Мария"];

Stack<double> stack = new Stack<double>();
stack.Push(2.2); stack.Push(3.3);
double top = stack.Pop();

Queue<int> queue = new Queue<int>();
queue.Enqueue(4); queue.Enqueue(5);
int firstOut = queue.Dequeue();
```

## Перебор, методы и сортировка

- Перебор массива или коллекции осуществляется с помощью цикла (обычно foreach).
- Для поиска элементов и сортировки используются встроенные методы и алгоритмы.

**Пример сортировки списка:**
```csharp
List<int> nums = new List<int> {5, 1, 9, 3};
nums.Sort(); // {1, 3, 5, 9}
```

## Практические советы
- Используйте массивы для фиксированных структур и списки — для динамических данных.
- Словари и хеш-таблицы оптимальны для хранения пар «ключ-значение».
- Добавляйте комментарии к алгоритмам для удобства чтения и поддержки кода.
