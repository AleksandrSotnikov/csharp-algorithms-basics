# Коллекции и LINQ в C#

В этом модуле рассматриваются современные подходы к работе с данными в .NET: коллекции, их отличия и структура, а также технологии обработки с помощью LINQ (Language Integrated Query).

## Стандартные коллекции .NET

- `List<T>`: динамический список, замена массивам при изменяющемся размере.
- `Dictionary<TKey, TValue>`: ассоциативный массив, быстрый поиск по ключу.
- `HashSet<T>`: уникальные значения без повторов.
- `Queue<T>`, `Stack<T>`: структуры для организации порядка обработки элементов.

**Краткие примеры:**
```csharp
List<int> numbers = new List<int> {1, 2, 5};
numbers.Add(8);
Dictionary<string,int> ages = new Dictionary<string,int>();
ages["Анна"] = 22;
HashSet<string> emails = new HashSet<string> {"a@mail.ru", "b@mail.ru"};
```

## Основы LINQ

LINQ — это универсальный язык запросов для работы с коллекциями, массивами, файлами, XML и базами данных непосредственно из языка C#.

- Позволяет фильтровать, сортировать, преобразовывать и группировать данные с минимальным кодом.

**Пример:**
```csharp
List<int> nums = new List<int> {1, 3, 2, 1, -5, 22};
var even = nums.Where(n => n % 2 == 0).ToList();
var desc = nums.OrderByDescending(n => n).ToList();
int max = nums.Max();
var sum2 = nums.Where(n => n > 0).Sum();
```

**Запросы LINQ к коллекции:**
```csharp
string[] langs = {"C#", "Java", "C++", "Python"};
var query = from l in langs
            where l.StartsWith("C")
            orderby l
            select l;
foreach(var lang in query) Console.WriteLine(lang);
```

## Методы расширения

LINQ реализован через методы расширения для всех коллекций, реализующих интерфейс IEnumerable<T>:
- `Where`, `Select`, `OrderBy`, `GroupBy`, `ToList`, `Any`, `All`, `FirstOrDefault` и многие другие.

## Практические упражнения
1. Сформируйте список сотрудников, выберите все имена, начинающиеся с определённой буквы.
2. Используйте Dictionary для хранения контактов (имя — телефон), реализуйте быстрый поиск по ключу.
3. Отберите из списка чисел только уникальные элементы и отсортируйте их по убыванию.
4. Для заданного массива строк найдите самую длинную и самую короткую с помощью LINQ.
5. Проведите группировку элементов (например, по первой букве) с использованием LINQ.

## Рекомендации
- LINQ позволяет делать сложные выборки с минимальным объёмом кода. Применяйте его к коллекциям всех типов.
- Всегда комментируйте поддерживающий функционал методов, если выборка неочевидна.
- Используйте ToList в конце цепочки для получения конкретного результата.
