# Функции (методы) в C#

В этом модуле изучаются механизмы разбиения кода на функциональные блоки, правила создания и использования методов, параметры передачи данных, возврат результата и принципы рекурсии.

## Теоретический материал

### Объявление и вызов методов

Метод — это именованный набор инструкций, которые можно многократно вызывать из разных частей программы. Метод объявляется обычно внутри класса, имеет возвращаемый тип, имя и список параметров.

```csharp
public int Sum(int a, int b) // объявление метода
{
    return a + b;
}

// Вызов метода
int result = Sum(2, 3); // 5
```

### Параметры методов

В C# параметры могут передаваться:
- по значению (обычно)
- по ссылке (`ref` и `out`)

```csharp
void Update(ref int x) // x может быть изменен внутри метода
{
    x = 100;
}
int a = 7;
Update(ref a); // a = 100

void GetInfo(out int value)
{
    value = 50;
}
int data;
GetInfo(out data); // data = 50
```

### Перегрузка и рекурсивные методы

Перегрузка: несколько методов с одинаковым именем, но разными параметрами.
```csharp
int Multiply(int a, int b) { return a * b; }
int Multiply(int a, int b, int c) { return a * b * c; }
```

Рекурсия: метод вызывает сам себя для решения задачи в несколько этапов.
```csharp
int Factorial(int n)
{
    if (n <= 1) return 1;
    return n * Factorial(n - 1);
}
```

## Практические рекомендации
- Следите за осмысленностью и читаемостью имен методов.
- Каждый метод должен выполнять одну логическую задачу — принцип единой ответственности.
- Для сложных алгоритмов добавляйте подробные комментарии к шагам вычислений.
