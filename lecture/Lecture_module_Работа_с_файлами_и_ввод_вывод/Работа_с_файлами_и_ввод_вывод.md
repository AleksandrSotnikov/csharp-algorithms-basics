# Работа с файлами и ввод-вывод в C#

Данный модуль посвящён основам работы с файлами и потоками, методам чтения и записи информации, сериализации/десериализации, асинхронному вводу-выводу.

## Теоретический материал

### Чтение и запись в текстовые файлы

Для взаимодействия с файлами в C# используются классы пространства имён System.IO:

```csharp
// Запись в файл
File.WriteAllText("test.txt", "Привет, мир!");

// Чтение из файла
string text = File.ReadAllText("test.txt");
Console.WriteLine(text);

// Построчное чтение
string[] lines = File.ReadAllLines("test.txt");
foreach (var line in lines) {
    Console.WriteLine(line);
}
```

### Работа с потоками ввода-вывода

Потоки позволяют работать с большими объёмами данных, файлами, сетевыми подключениями:
```csharp
using (var sw = new StreamWriter("data.log")) {
    sw.WriteLine("Журнал работы");
}

using (var sr = new StreamReader("data.log")) {
    string info = sr.ReadToEnd();
}
```

### Сериализация и десериализация

Сериализация — превращение объектов в поток данных (byte/string), который можно записать или переслать:
```csharp
// Пример сериализации JSON
Person p = new Person { Name = "Иван", Age = 20 };
string json = JsonSerializer.Serialize(p);
File.WriteAllText("p.json", json);

// Десериализация
string raw = File.ReadAllText("p.json");
Person restored = JsonSerializer.Deserialize<Person>(raw);
```

### Асинхронная работа с файлами

Асинхронное выполнение операций делает программу отзывчивой и быстрой:
```csharp
await File.WriteAllTextAsync("async.txt", "Асинхронный режим!");
string content = await File.ReadAllTextAsync("async.txt");
```

## Практические рекомендации
- Всегда проверяйте существование файла перед чтением
- Используйте блоки using для автоматического освобождения ресурсов
- Обрабатывайте исключения: FileNotFoundException, IOException

## Практические задания
1. Запишите в текстовый файл массив строк и загрузите обратно.
2. Сериализуйте и десериализуйте объект через JSON.
3. Реализуйте простой логгер событий, который сохраняет информацию в файл в реальном времени.
4. Используйте асинхронные операции для чтения/записи файла большого размера.
