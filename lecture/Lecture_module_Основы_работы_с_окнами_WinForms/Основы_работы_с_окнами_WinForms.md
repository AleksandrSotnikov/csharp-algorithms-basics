# Основы работы с окнами (WinForms) в C#

Данный модуль посвящён построению графических интерфейсов (GUI) с использованием технологии Windows Forms, их архитектуре, созданию и обработке элементов управления.

## Теоретическая основа

### Понятие WinForms

Windows Forms — часть платформы .NET Framework, позволяющая создавать оконные (визуальные) приложения с элементами управления: кнопками, текстовыми полями, списками и т.д.

- Приложение WinForms состоит из форм (`Form`), которые содержат визуальные компоненты (controls).

### Создание проекта WinForms

1. Запустите Visual Studio, выберите шаблон "Windows Forms Application (.NET Framework)"
2. Дайте имя проекту и выберите расположение.
3. Откроется дизайнер форм: добавляйте кнопки, текстовые поля, списки из панели Toolbox.
4. Настраивайте свойства элементов в окне Properties (Name, Text, Size, Location, и др.).

### Кодовое взаимодействие

Каждый элемент управления можно программно создавать и настраивать:

```csharp
public partial class MainForm : Form {
    public MainForm() {
        InitializeComponent();
        Button btn = new Button();
        btn.Text = "Нажми меня";
        btn.Click += Btn_Click;
        this.Controls.Add(btn);
    }
    private void Btn_Click(object sender, EventArgs e) {
        MessageBox.Show("Кнопка нажата!");
    }
}
```

### События и обработчики

- Любое действие пользователя (клик, ввод текста, изменение значения) генерирует событие.
- Для обработки назначаются методы-обработчики (например, Click, TextChanged).
- Связывание события с обработчиком происходит через оператор `+=`.

### Диалоги и работа с файлами

В WinForms доступны стандартные диалоговые окна (FileDialog, ColorDialog, MessageBox). Пример открытия файла:

```csharp
OpenFileDialog dlg = new OpenFileDialog();
if (dlg.ShowDialog() == DialogResult.OK) {
    string path = dlg.FileName;
    MessageBox.Show("Файл открыт: " + path);
}
```

## Практические рекомендации
- Соблюдайте логику иерархии форм: главная форма, дочерние окна/диалоги.
- Каждый элемент форм описывайте подробно: назначение, событие, обработка.
- Для сложных интерфейсов используйте панель инструментов, GroupBox, панели и т.д.

## Практические задания
1. Создайте простое приложение c формой, полем ввода и кнопкой. При нажатии выводите содержимое поля в MessageBox.
2. Реализуйте список с элементами и возможность их добавления/удаления.
3. Используйте OpenFileDialog для загрузки и вывода содержимого выбранного файла.
