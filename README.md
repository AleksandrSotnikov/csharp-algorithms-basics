# C# Основы Алгоритмизации

Репозиторий с примерами и реализациями основных алгоритмов на C#.

## Структура репозитория

### 1. Базовые алгоритмы
- Линейный поиск
- Бинарный поиск
- Пузырьковая сортировка
- Сортировка вставками
- Сортировка выбором

### 2. Алгоритмы сортировки
- Быстрая сортировка (QuickSort)
- Сортировка слиянием (MergeSort)
- Пирамидальная сортировка (HeapSort)

### 3. Структуры данных
- Массивы
- Связные списки
- Стек
- Очередь
- Хеш-таблицы
- Деревья

### 4. Рекурсия
- Факториал
- Числа Фибоначчи
- Обход дерева

### 5. Алгоритмы на графах
- Поиск в ширину (BFS)
- Поиск в глубину (DFS)
- Алгоритм Дейкстры

### 6. Динамическое программирование
- Задача о рюкзаке
- Наибольшая общая подпоследовательность

## Как использовать

1. Клонируйте репозиторий:
```bash
git clone https://github.com/AleksandrSotnikov/csharp-algorithms-basics.git
```

2. Откройте проект в Visual Studio или VS Code

3. Изучайте примеры и запускайте код

## Требования

- .NET 6.0 или выше
- Visual Studio 2022 или VS Code
