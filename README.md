# Основы программирования на языке C# (.NET Framework 4.8)

Этот репозиторий предназначен для изучения фундаментальных тем программирования на языке C# с детализацией каждой темы и актуальной версией .NET Framework 4.8.

## Темы для изучения

### 1. Введение в C# и .NET Framework
- Обзор языка C#, его история и назначение
- Особенности платформы .NET Framework 4.8
- Структура программ на C#, типы проектов
- Установка и настройка среды разработки (Visual Studio)

### 2. Синтаксис и базовые конструкции
- Объявление, инициализация переменных
- Типы данных: простые, ссылочные
- Константы, преобразования типов
- Операторы (арифметические, логические, побитовые, сравнения)
- Комментарии в коде

### 3. Управляющие конструкции
- Условные операторы: if, else, switch
- Операторы цикла: for, while, do while, foreach
- Операторы перехода: break, continue, return, goto

### 4. Массивы и коллекции
- Одномерные, двумерные, многомерные массивы
- Стандартные коллекции: List, Dictionary, Stack, Queue
- Индексация, перебор, методы работы с коллекциями

### 5. Функции (Методы)
- Объявление и вызов методов
- Параметры, возвращаемые значения
- Передача параметров по значению и по ссылке (ref, out)
- Перегрузка методов
- Рекурсивные методы

### 6. Классы и объекты
- Понятие класса, создание объектов
- Свойства, поля, методы класса
- Конструкторы и их типы
- Модификаторы доступа (public, private, protected, internal)
- Статические члены класса
- Деструкторы

### 7. Наследование и полиморфизм
- Наследование классов, базовые и производные классы
- Виртуальные и абстрактные методы/классы
- Переопределение методов
- Интерфейсы, реализация интерфейсов
- Полиморфизм: compile-time и run-time

### 8. Обработка ошибок и исключения
- Основы try-catch-finally
- Типы исключений
- Генерация и обработка пользовательских исключений
- Практика отладки: точки остановки, просмотр стека вызовов

### 9. Работа с файлами и ввод-вывод
- Чтение и запись в текстовые файлы
- Работа с потоками ввода-вывода
- Сериализация и десериализация данных
- Асинхронная работа с файлами

### 10. Основы работы с окнами (WinForms)
- Введение в WinForms
- Создание простых оконных приложений
- Элементы управления: кнопки, текстовые поля, списки
- События, обработка пользовательского ввода
- Диалоги и файловые окна

### 11. Коллекции и LINQ
- Стандартные коллекции .NET
- LINQ: запросы, методы расширения, фильтрация и сортировка данных
- Примеры взаимодействия с коллекциями через LINQ

### 12. Делегаты и события
- Понятие делегата, типы делегатов
- Анонимные методы и лямбда-выражения
- События, подписка и обработка событий

### 13. Основы многопоточности
- Потоки в .NET, класс Thread
- Пул потоков
- Асинхронное программирование: async/await
- Проблемы синхронизации, понятие deadlock

### 14. Принципы ООП и шаблоны проектирования
- Инкапсуляция, наследование, полиморфизм
- Простые шаблоны: Singleton, Factory, Observer, Strategy

### 15. Практические проекты и упражнения
- Практическая реализация тем в виде мини-приложений
- Задачи для закрепления материала

## Как использовать

1. Клонируйте репозиторий:
```bash
git clone https://github.com/AleksandrSotnikov/csharp-algorithms-basics.git
```
2. Откройте проект в Visual Studio 2019/2022 (или другой IDE, поддерживающей .NET Framework 4.8)
3. Изучайте темы и запускайте демонстрационные примеры

## Требования

- .NET Framework 4.8
- Visual Studio 2019/2022
