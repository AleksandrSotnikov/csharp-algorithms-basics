# Структура репозитория

Для более комфортного изучения и систематизации материалов добавлены две ключевые папки:

## 1. lecture

- `lecture/content.md` — оглавление лекций.
- Каждая лекция располагается в отдельной папке вида `Lecture_module_Название_темы`.
- Внутри — «[translate:Подробная лекция в научно публицистическом стиле]». Материал лекции структурирован, содержит разъяснения, научные термины, практические примеры.

## 2. task

- `task/content.md` — оглавление практических заданий.
- Каждое задание размещается в папке по теме (например, `Task_Название_темы`).
- Внутри —
    - Файл автоматизированного тестирования: `autotest.cs` — покрывает проверки корректности выполнения задания.
    - Файл `assignment.md` — инструкция для выполнения задания с подробными условиями, где студент ведёт работу и отвечает на вопросы.

## Дополнительно

- Для самостоятельной работы предусмотрена папка `selfstudy` — дополнительные задачи, интересные материалы, ссылки на источники для углубления.
- Папка `examples` — содержит краткие рабочие примеры по темам для быстрой практики.

## Как начать работу

1. Ознакомьтесь с содержанием в файле `lecture/content.md`
2. Изучайте лекции по модулям — каждый материал в отдельной папке по теме
3. Переходите к заданиям в папке `task`: ознакомьтесь с условиями, выполняйте решение, проверяйте автоматизированными тестами

## Рекомендации для студентов

- Ведите конспекты, отмечайте важные моменты в лекционных файлах
- Старайтесь выполнять все задания для закрепления материала
- Используйте дополнительные материалы из папки `selfstudy` для расширения кругозора
